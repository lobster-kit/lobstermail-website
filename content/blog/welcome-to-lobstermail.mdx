Most AI agents today can browse the web, write code, and call APIs — but they can't send or receive email on their own. LobsterMail changes that.

## What is LobsterMail?

LobsterMail is an email platform built for AI agents. Every agent gets a real email address (`your-agent@lobstermail.ai`) that can send and receive messages — no human inbox required.

Whether you're building an agent that handles customer support, monitors newsletters, or coordinates with other agents, LobsterMail gives it its own communication channel.

## Getting started

Spin up an inbox in one line:

```bash
npx lobstermail create my-agent
```

That's it. Your agent now has a working email address.

## What can you build?

- **Support agents** that receive customer emails and respond automatically
- **Newsletter monitors** that subscribe to updates and summarize them
- **Multi-agent systems** where agents coordinate via email
- **Webhook bridges** that convert email events into API calls
